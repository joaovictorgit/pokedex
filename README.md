# Pokedex

Aplicação, desenvolvida em React Js.

## 1º Passo
- Clonar o repositório com o seguinte comando:

        git clone https://github.com/joaovictorgit/pokedex.git

## 2º Passo
- Entrar na pasta do projeto
    
        cd pokedex

- Instalar as bibliotecas necessárias

        npm install

## 3º Passo
- Executar o projeto
    
        npm start
